# 运算符

## 指数运算符（`**`）

指数运算符（`**`）：
指数运算符的特点是右结合，多个指数运算符连用时，是从最右边开始计算的,eg：

```
2 ** 3 ** 2 | 2 ** (3 ** 2)
```

## 链判断运算符（`?.`）

`?.`运算符，直接在链式调用的时候判断，左侧的对象是否为 null 或 undefined。如果是的，就不再往下运算，而是返回 undefined

链判断运算符?.的三种写法

-   `obj?.prop` // 对象属性是否存在
-   `obj?.[expr]`// 对象属性是否存在
-   `func?.(...args)` // 函数或对象方法是否存在
