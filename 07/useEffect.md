# useEffect Hooks

### 参数1:箭头函数()=>{}，在真正渲染 html 之前会执行它

### 参数2:

* 情况1:没有，代表每次执行组件函数时，都会执行副作用函数
```
useEffect(() =>{} )
```
* 情况2:[]空数组，代表副作用函数只会执行一次

```
useEffect(() =>{},[] )
```
* 情况3:[依赖项]，依赖项变化时，副作用函数会执行

```
useEffect(() =>{},[依赖项] )
```
# useState Hooks

参数:数据的初始值
返回值:[a,b]
a:状态数据
b:方法,修改状态数据的方法setXX()
```
const [a,b] = useState(初始值)
```
